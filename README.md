# Rekomendasi Aktivitas Mengatasi Stress dan Meningkatkan Produktivitas Manusia - Capstone Project SIB x Dicoding - Team C22-127

Proyek ini dirancang untuk membangun website yang dapat digunakan untuk menghasilkan keluaran berupa rekomendasi kegiatan atau aktivitas berdasarkan mood yang dirasakan oleh pengguna.
